## Actors and Their Interactions

### Administrator

The Administrator is responsible for managing, configuring, and supervising the continuous monitoring and alert automation system.

#### Interactions
- Admin Login
- View Dashboard  
- Monitor System Metrics  
- Configure Monitoring Rules and Thresholds  
- View Data and Alert History  
- Role Management
- Alert Generation
- Sending Alerts
- Escalate Alerts

The Administrator ensures the system operates reliably, maintains system health, manages user roles, and handles alert escalation when abnormal conditions are detected.

### Unregistered Users

The UnRegistered Users are the Users who dont have their account in this system

#### Interactions
- User Registration
- DashBoard View
