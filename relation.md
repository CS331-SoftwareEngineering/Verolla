## Include Relations

<<include>> means the base use case always uses another use case as part of its execution.

# Detect Anomalies <<include>> Generate Alerts

Whenever an anomaly is detected, alert generation is a mandatory step.

# Detect Anomalies <<include>> Send Notifications

After detecting anomalies, notifications must be sent to users or systems.

# Generate Alerts <<include>> Send Notifications

Every generated alert must be communicated via notifications.

# Log Events & Alerts <<include>> Send Notifications

 Logging of events and alerts includes notifying relevant stakeholders.
